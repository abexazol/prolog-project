Along with this file, you should have received two other
files:

	- A winrar-archive containing the source code
	- A report in pdf-format

To run the source code, it suffices to load the file 'main.pl'
in a SWI-prolog environment. Typing

	main(_PATH_TO_FILE_).

in the terminal, where _PATH_TO_FILE_ is a string containing the
path to some example file, like: 'examples/2pairs.txt'. Swipl
will then print some output, and return true, if the file was
found.
